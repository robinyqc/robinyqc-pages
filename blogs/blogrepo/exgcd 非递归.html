exgcd 非递归

exgcd 大家都会，gcd 的非递归写法（甚至 binary gcd,基于值域的 gcd）大家也都会。但为什么 exgcd 的非递归写法我没听说过呢？原来大家也都会，只是我孤陋寡闻罢了。

考虑方程组：

{ax+by=cdx+ey=f

或者说（以下都用这种方式表示）：

(abcdef)

用你最喜欢的消项法解方程：

(abcdef)⟹(abcd−pae−pbf−pc)

令 p=⌊fc⌋，则：

(abcd−pae−pbf−pc)=(abcd−pae−pbcmodf)

发现常数项就是 gcd 的过程。那么对任一方程组 (abcdef)，必可以通过若干变换变成 (nmgcd(c,f)ij0)。

那么对应 exgcd 的过程就是：

原问题：求解 ax+by=gcd(a,b)。

我们把 a,b 当成主元。初始方程： (10a01b)

根据上面的变换方式，必能得到等价矩阵 (nmgcd(a,b)ij0)。就解出来了，因为矩阵的第一个行向量，就是表示的 na+mb=gcd(a,b)。答案就是 x=n,y=m。

代码：

int exgcd(int a,int b,int &x,int &y) {
    x=1,y=0;
    int n=0,m=1,t;
    while(b) {
        t=n,n=x-a/b*n,x=t;
        t=m,m=y-a/b*m,y=t;
        t=b,b=a%b,a=t;
    }
    return a;
}
